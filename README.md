# Suduq

Suduq is a fast browser-first utility suite for image, text, developer, AI, and utility workflows.

## Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Express + Multer + Sharp
- AI: Google AI / Gemini

## First sprint

- Image compress
- Image resize
- Image convert
- Image to base64

## Getting started

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:5173` and the API runs on `http://localhost:4000`.
